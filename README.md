# 🚀 DeltA Digivast – Frontend

🌐 Live URL: https://delt-a-digivast-frontend.vercel.app/

---

## 📌 Project Overview

DeltA Digivast is a modern **Agency Website + Office Management System** built
with a powerful frontend stack.

This platform includes:

- Agency website (Landing, Services, Portfolio, Blog)
- Internal office management system
- Staff dashboard & attendance tracking
- Course purchasing system (Stripe integration)

---

## 🛠️ Tech Stack

- ⚛️ React (Next.js)
- 📦 pnpm
- 🔷 TypeScript
- 🎨 Tailwind CSS
- 🧩 shadcn/ui
- 🔐 JWT Authentication
- 💳 Stripe Payment Integration

---

## 📂 Project Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/MAHIR-DEVES/DeltA-Digivast-Frontend
cd DeltA-Digivast-Frontend

2️⃣ Install dependencies
pnpm install

3️⃣ Setup environment variables

Create a .env.local file:

NEXT_PUBLIC_API_URL=https://delt-a-digivast-backend.vercel.app/
NEXT_PUBLIC_IMGBB_API_KEY=yourNEXT_PUBLIC_IMGBB_API_KEY

4️⃣ Run the project
pnpm dev
```
